Injected Wallet

Live wallet path for connect, sign, quote, submit, and cancel through the root `cow-sdk` surface. Detect caches reviewed wallet candidates, Confirm Wallet records the provider to use when more than one candidate is present, Connect / Reconnect uses the confirmed provider or the retained wallet handle, Rescan refreshes the candidate set while revalidating the confirmed choice, Reset Session keeps the selected wallet, and Forget Wallet clears both the selected wallet and the confirmed provider from the console.
