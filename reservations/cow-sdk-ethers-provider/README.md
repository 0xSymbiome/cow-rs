# cow-sdk-ethers-provider (reserved)

This crate name is reserved as an optional ethers-rs provider adapter
for the
[CoW Protocol Rust SDK](https://github.com/0xSymbiome/cow-rs),
delivered under the CoW DAO Community Grants Program.

## Status

This `0.0.1-reserved.0` pre-release exists solely to reserve the crate
name on crates.io while development is underway. No functionality is
implemented at this version.

## Planned Release

When published as `0.1.0` or later, this crate will provide an optional
adapter that implements the cow-sdk-core runtime trait contracts over
the ethers-rs web3 library, complementing the primary
`cow-sdk-alloy-provider` crate.
