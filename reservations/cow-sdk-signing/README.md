# cow-sdk-signing (reserved)

This crate name is reserved for the order signing primitives of the
[CoW Protocol Rust SDK](https://github.com/0xSymbiome/cow-rs),
delivered under the CoW DAO Community Grants Program.

## Status

This `0.0.1-reserved.0` pre-release exists solely to reserve the crate
name on crates.io while development is underway. No functionality is
implemented at this version.

## Planned Release

When published as `0.1.0` or later, this crate will implement EIP-712
typed-data signing, EIP-1271 smart-contract signature verification
helpers, and the signing-error taxonomy used by the trading and
orderbook crates.
