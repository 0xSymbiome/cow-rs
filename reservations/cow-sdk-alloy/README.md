# cow-sdk-alloy (reserved)

This crate name is reserved for the composed alloy-rs provider and
signer adapter of the
[CoW Protocol Rust SDK](https://github.com/0xSymbiome/cow-rs),
delivered under the CoW DAO Community Grants Program.

## Status

This `0.0.1-reserved.0` pre-release exists solely to reserve the crate
name on crates.io while development is underway. No functionality is
implemented at this version.

## Planned Release

When published as `0.1.0` or later, this crate will bundle the alloy-rs
provider and signer adapters into a single convenience facade that
implements the cow-sdk-core runtime trait contracts over the alloy Rust
web3 library.
