# Examples

The examples are part of the public proof surface for `cow-rs`.

## Native

`examples/native/scenarios/` covers deterministic and mocked Rust flows for app-data, signing, orderbook, trading, subgraph, and facade usage.

## WASM

`examples/wasm/sdk-verification-console/` proves the SDK build and browser-facing surface.

`examples/wasm/browser-wallet-console/` proves mock-wallet and injected-wallet browser flows through the public SDK.

## Role

Keep the top-level docs short. Let the example directories carry the detailed walkthroughs.
